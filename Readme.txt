才華有很多種，每個人都有自己獨有的長處，近日一名來自澳洲的女子麥基夫（Beatrice McKeefe）就前往德國參加了一檔選秀節目，大秀超狂「才華」，她不僅倒立、雙腿開開，甚至將直笛插入下體吹奏起「Do Re Mi」。現場的觀眾被女子的「才華」嚇得驚呼連連，狂喊OMG。

綜合外媒報導，這名來自澳洲的麥基夫平常兼任外科助理，因對體操、馬戲團表演有興趣，因此遠赴德國參加一檔選秀節目。沒想到，她竟在現場中以非常獨特的方式演奏直笛，她現是將直笛插入自己的下體，接著倒立在兩張桌子之間，兩腳開開吹奏起「Do Re Mi」，現場的評審紛紛被眼前這一幕給嚇傻，就連主持人也大吃一驚，現場觀眾有人摀嘴、有人害羞遮臉，狂喊OMG。

不過，這樣的舉動卻引來了不少聲量，就有網友認為這種表演並不適合播出，因為現場有小孩和未成年的觀眾。據了解，這並非麥基夫首次展露這項技能，去年9月她同樣也在克羅埃西亞的選秀節目上，表演用下體吹奏直笛。
